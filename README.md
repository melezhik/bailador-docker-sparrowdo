# SYNOPSIS

The example of running simple Bailador application using docker/sparrowdo

# USAGE

## Checkout the project

    $ git clone

## Build you image 

    $ docker build -t bailador .

## Run application

    $ docker run $image_id

# Project structure

## example.p6w

Simple bailador application

## sparrowfile

Sparrowdo scenario to deploy Bailador 

## entrypoint.sh

Default entrypoint
